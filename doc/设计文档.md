# 设计文档

## 总体设计
### v0.1
1. 访问网站 (c++ 爬虫)
2. 拉取图片 (单张->多张)
3. 存储图片 (文件操作)

### v0.2
加入分发器 Dispatcher
1. Visitor 负责访问网站并返回
    - 重定向地址
    - 成功响应中的 html 段
    - 图片编码
2. Response_Parser 负责解析 Visitor 返回的成功响应中的 html 段，提取 jpg 文件地址
3. Dispatcher 负责接收地址并针对每个地址单独生成 Visitor 进行访问
4. IMGWriter 负责存储 Visitor 返回的图片编码
